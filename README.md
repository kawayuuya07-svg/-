# 今日の1% - 習慣化PWA

Atomic Habits の考え方をもとにした、シンプルな習慣化ツールです。

## できること
- 今日の習慣チェック
- 1〜5分の開始タイマー
- 連続日数、今週の実行率、直近28日の記録
- 習慣の追加・編集・削除
- アイデンティティ文の編集
- ダークモード
- データの書き出し / 読み込み
- オフライン利用（PWA）

## 使い方
1. `index.html` をブラウザで開く
2. iPhoneの場合はSafariで開いて、共有ボタン → 「ホーム画面に追加」
3. Android/Chromeの場合はメニュー → 「アプリをインストール」

## 注意
データは端末内の localStorage に保存されます。機種変更やブラウザ変更の前に、設定画面からデータを書き出してください。
