# 今日の1% - 習慣化PWA v2

日々の積み重ねが残る習慣化ツールです。

## v2で改善したこと
- 日付ごとの記録を蓄積
- その日に存在していた習慣内容をスナップショット保存
- 習慣の追加・編集・アーカイブ履歴に対応
- 削除済み習慣も過去の積み重ねに反映
- 累計完了回数、記録日数、最長連続を表示
- 日別履歴を表示
- 旧版データの自動移行を試行

## GitHub Pagesで使う場合
以下をリポジトリ直下に置いてください。

- index.html
- manifest.webmanifest
- sw.js
- icon.svg
- README.md

## 注意
データは端末内の localStorage に保存されます。GitHubには保存されません。機種変更前は「データを書き出す」でバックアップしてください。
